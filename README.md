# Student Record Management System

## 📌 Project Overview
The Student Record Management System is a robust, menu-driven backend application designed to efficiently manage, store, and retrieve student data. Built with a strong focus on Object-Oriented Programming (OOP) principles, this system allows administrators to seamlessly perform CRUD (Create, Read, Update, Delete) operations on student profiles.

This project demonstrates practical implementations of core computer science concepts, including dynamic data structures, algorithmic search mechanisms, and system architecture.

## 🚀 Features
*   **Add New Records:** Securely input and instantiate new student profiles with attributes such as Roll Number, Name, and GPA.
*   **View Database:** Dynamically iterate through the stored data structure to display all current student records.
*   **Search Algorithms:** Efficiently locate specific student records using designated identifiers (e.g., Roll Number).
*   **Modify & Delete:** Update existing student information or securely remove records from the database, demonstrating dynamic memory management.
*   **Interactive UI:** Features a continuous, menu-driven command-line interface for intuitive user interaction.

## 💻 Tech Stack
*   **Language:** C++
*   **Core Concepts Used:** Object-Oriented Programming (Classes, Objects, Encapsulation), Dynamic Data Structures (Vectors), Control Flow.
